I. Pendahuluan & Konsep Dasar

1.1 Pengantar Kerangka Kontrol Geodesi

Definisi

Kerangka Kontrol Geodesi adalah jaringan titik-titik tetap yang koordinatnya diketahui secara teliti dan menjadi referensi utama seluruh pekerjaan survei dan pemetaan.

Tujuan Pengukuran

  • Akurasi = dekat dengan nilai sebenarnya
  • Presisi = hasil konsisten berulang

Fungsi KKG

  • Fondasi Sistem Informasi Geografis
  • Dasar pembangunan jalan, jembatan, gedung
  • Acuan batas bidang tanah / kadaster
  • Referensi deformasi dan monitoring
Contoh:
Titik BM nasional digunakan sebagai referensi survei jalan tol.
I. Pendahuluan & Konsep Dasar

1.2 Mengapa Perlu Perataan?

Karena terdapat redundansi pengukuran. Misalnya titik A ke B diukur 3 kali:

Data Pengukuran

  • 100.024 m
  • 100.019 m
  • 100.027 m
\[ \bar{x}=\frac{100.024+100.019+100.027}{3}=100.0233 \, m \]

Jenis Kesalahan

  1. Blunder → salah baca / salah catat
  2. Sistematis → koreksi suhu, atmosfer
  3. Acak → fluktuasi kecil

Tujuan Adjustment

  • Mendapat nilai terbaik
  • Mengurangi pengaruh error acak
  • Menentukan ketelitian hasil
II. Teori Kesalahan dan Prinsip Perataan

2.1 Distribusi Kesalahan

Kesalahan acak dalam pengukuran geodesi umumnya diasumsikan mengikuti distribusi normal.

\[ f(x)=\frac{1}{\sigma\sqrt{2\pi}}e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2} \]

Keterangan

  • \(\mu\) = nilai rata-rata
  • \(\sigma\) = simpangan baku

Makna dalam Geodesi

  • Error kecil sering terjadi
  • Error besar jarang terjadi
  • Digunakan untuk analisis residual
  • Digunakan dalam uji statistik
II. Teori Kesalahan dan Prinsip Perataan

2.2 Metode Kuadrat Terkecil

Prinsip Utama

\[ \sum v_i^2 = minimum \]

Artinya, nilai parameter dicari sehingga jumlah kuadrat selisih antara hasil pengamatan dan nilai hasil hitungan menjadi sekecil mungkin.

Contoh:
Jarak hasil ukur: 100.10 ; 100.00 ; 99.95

\[ x=\frac{100.10+100.00+99.95}{3}=100.0167 \] Residual:
  • +0.0833
  • -0.0167
  • -0.0667
III. Perataan Jaringan Kerangka Kontrol

3.1 Model Parameter

\[ Ax = l + v \]

Keterangan

  • \(A\) = matriks desain
  • \(x\) = parameter tak diketahui
  • \(l\) = observasi
  • \(v\) = residual

Solusi Least Squares

\[ \hat{x}=(A^TPA)^{-1}A^TPl \]

\(P\) adalah matriks bobot pengamatan.

Contoh: Menentukan koordinat titik baru dari pengamatan sudut dan jarak.
III. Perataan Jaringan Kerangka Kontrol

3.2 Ketelitian dan Error Ellipse

Koordinat hasil adjustment memiliki kovariansi:

\[ \Sigma_x = \sigma_0^2(A^TPA)^{-1} \]

Makna

Matriks kovariansi digunakan untuk mengetahui presisi koordinat titik hasil perataan.

Error Ellipse

  • Sumbu mayor = 5 mm
  • Sumbu minor = 2 mm
  • Semakin kecil elips, semakin teliti posisi titik.
Error ellipse membantu menjelaskan ketidakpastian posisi dalam bentuk visual dua dimensi.

V. Penutup

Kesimpulan

  • Kerangka Kontrol Geodesi adalah dasar seluruh survei presisi.
  • Semua pengukuran pasti mengandung error.
  • Adjustment diperlukan agar data konsisten.
  • Least Squares adalah metode utama geodesi modern.
  • Hasil akhir bukan hanya koordinat, tetapi juga ketelitiannya.
